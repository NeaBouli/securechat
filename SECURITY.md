# Security Policy

## Scope

This security policy covers:

- **SecureChat website** — static site and documentation at securechat.stealthx.tech
- **SecureChat Android app** — future encrypted messaging application (Phase 1, Q2-Q3 2026)
- **Crypto implementation** — XChaCha20-Poly1305, X25519, Double Ratchet, Argon2id, Ed25519 via lazysodium-android
- **Chameleon** — overlay encryption component

## Reporting Vulnerabilities

**Please report security vulnerabilities via [GitHub Security Advisories](https://github.com/NeaBouli/securechat/security/advisories/new).**

This ensures your report is handled privately and securely. Do not open a public issue for security vulnerabilities.

If you are unable to use GitHub Security Advisories, you can reach us at:

- **Email:** kaspartisan@proton.me
- **GitHub Issues:** For non-sensitive security questions only

## Disclosure Policy

- We follow a **90-day embargo** on public disclosure from the time a vulnerability is reported.
- We will acknowledge receipt of your report within 48 hours.
- We will provide an initial assessment within 7 days.
- We will work with you to understand and resolve the issue before any public disclosure.
- Credit will be given to the reporter in the security advisory (unless anonymity is requested).

## Bug Bounty

There is no formal bug bounty program at this time. We appreciate responsible disclosure and will credit all valid reports.
